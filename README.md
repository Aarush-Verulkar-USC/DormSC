# 🏠 DormSC

Student housing platform for USC — find verified off-campus listings near campus.

## ✨ Features

- 🗺️ Interactive Google Maps with route-to-campus
- 🔍 Search & filter by price, beds, baths
- ❤️ Save favorite listings
- 🔐 Firebase auth (USC email verified)
- 📋 List, edit & manage your own properties
- 🛡️ Admin dashboard for moderation

## 🛠️ Tech Stack

**Next.js** · **TypeScript** · **Tailwind CSS** · **Firebase** · **Google Maps API**

## 🚀 Getting Started

```bash
npm install
npm run dev
```

Create a `.env.local` with your Firebase and Google Maps API keys.

## 📦 Deploy

Hosted on [Vercel](https://vercel.com). Push to `main` to deploy.
